# Conventions: Commit Messages
___
## Summary
Try to do things right. Create commits as if you are working for an actual company. Be professional in all you do.

## Structure

```text
<type>[optional scope]: <description>

[optional body]

[optional footer(s)]
```

2. Feat
3. Build:
